Icon

아이콘 - name속성안에 사용할 아이콘의 종류를 선언하면 된다.

name="chat-left-dots" ->

아이콘의 크기 - 부모dom의 font-size를 따라간다
부모div의 font-size : 2rem ->
아이콘의 컬러 - 원래 Shoelace의 아이콘은 부모의 Color를 따라간다.
하지만 그럴경우 sideEffect가 너무 많아져서 CSS로 제어하였다.

단, 컬러의 지정이 필요할경우 -> 직접 sl-icon 에 style로 지정해주면 가능하다.

Custom Icon : name대신에 src=""를 사용해주면 된다.

Icon Button

Icon Button : name을 사용하는것은 sl-icon과 동일하나
sl-icon-button으로 선언
Icon Button 크기 : Icon 크기를 정할때와 같이 부모의 font-size 를 따라간다. 혹은 직접 지정해줘도 된다
inline 1.5rem -> div 2rem ->
Link Icon Button - href 속성과 target 속성 사용
Icon Tooltip - < sl-tooltip >으로 감싸준다
Disabled - disabled 속성을 사용한다.
