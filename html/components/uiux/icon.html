Icon - Shoelace에서 제공하는 icon 링크 ->

아이콘 - name속성안에 사용할 아이콘의 종류를 선언하면 된다.

name="chat-left-dots" -> 


    <sl-icon name="chat-left-dots"></sl-icon>
아이콘의 크기 - 부모dom의 font-size를 따라간다
부모div의 font-size : 2rem -> 

    <div style="font-size : 2rem">
        <sl-icon name="chat-left-dots"></sl-icon>
    </div>
아이콘의 컬러 - 원래 Shoelace의 아이콘은 부모의 Color를 따라간다.
하지만 그럴경우 sideEffect가 너무 많아져서 CSS로 제어하였다.

단, 컬러의 지정이 필요할경우 -> 직접 sl-icon 에 style로 지정해주면 가능하다.


    <div style="display: flex; gap: 1.5rem; font-size: 1.5rem; margin-top: 1.5rem;">
        <sl-icon name="chat-left-dots" style="color: red;"></sl-icon>
        <sl-icon name="chat-left-dots" style="color: blue;"></sl-icon>
        <sl-icon name="chat-left-dots" style="color: green;"></sl-icon>
    </div>
Custom Icon : name대신에 src=""를 사용해주면 된다. 

    <sl-icon src="https://shoelace.style/assets/images/shoe.svg" style="font-size: 4rem;"></sl-icon>

Icon Button

Icon Button : name을 사용하는것은 sl-icon과 동일하나
sl-icon-button으로 선언

    <sl-icon-button name="gear" label="Settings"></sl-icon-button>
Icon Button 크기 : Icon 크기를 정할때와 같이 부모의 font-size 를 따라간다. 혹은 직접 지정해줘도 된다
직접지정 1.5rem -> 부모 2rem -> 

    <div style="display: flex; gap: 1rem; align-items: flex-end;">
    직접지정 1.5rem ->
        <sl-icon-button name="pencil" label="Edit" style="font-size: 1.5rem;"></sl-icon-button>
        부모 2rem ->
        <div style="border: 1px solid lightgray; border-radius: 5px; font-size: 2rem; width: fit-content;">
            <sl-icon-button name="pencil" label="Edit" ></sl-icon-button>
        </div>
    </div>
Link Icon Button - href 속성과 target 속성 사용

    <sl-icon-button name="gear" label="Settings" href="https://example.com" target="_blank"></sl-icon-button>
Icon Tooltip - < sl-tooltip >으로 감싸준다

    <sl-tooltip content="Settings">
        <sl-icon-button name="gear" label="Settings"></sl-icon-button>
    </sl-tooltip>
Disabled - disabled 속성을 사용한다.

    <sl-icon-button name="gear" label="Settings" disabled></sl-icon-button>
