기초적인 Tooltip

사용법 : tooltip을 사용하려면 그냥 tooltip이 필요한 요소의 부모로
sl-tooltip을 사용하면된다.


tooltip에는 content="텍스트" 속성을 사용한다.

Hover Me 

  <sl-tooltip content="이곳에 보여질 텍스트를 넣는다">
      <sl-button>Hover Me</sl-button>
  </sl-tooltip>

Tooltip이 나타나는 trigger를 변경하고싶다면 trigger=""속성을 사용한다.

기본 trigger는 hover이다.

trigger="hover" trigger="click" 

  <sl-tooltip trigger="hover" content="tooltip">
      <sl-button>trigger="hover"</sl-button>
  </sl-tooltip>

  <sl-tooltip trigger="click" content="tooltip">
      <sl-button>trigger="click"</sl-button>
  </sl-tooltip>

  <sl-tooltip trigger="focus" content="tooltip" placement="top-start">
      <sl-input placeholder="trigger='focus'"></sl-input>
  </sl-tooltip>
특정 조건으로 tooltip을 활성화 시키기 -> JS필요 + trigger="manual" 속성 필요

tooltip이 가지는 .open 속성을 변경해줍니다.

툴팁 오픈! 

  <sl-button id="manual-button">툴팁 오픈!</sl-button>

  <sl-tooltip content="안녕하세요!" trigger="manual" id="manual-tooltip">
      <sl-avatar label="User"></sl-avatar>
  </sl-tooltip>

  <script>
      const button = document.getElementById("manual-button")
      const tooltip = document.getElementById("manual-tooltip")

      button.addEventListener("click", (e) => {
          tooltip.open = !tooltip.open;
      })
  </script>
tooltip의 화살표 없애기 - Style 수정 필요
No Arrow 

  <sl-tooltip content="This is a tooltip" style="--sl-tooltip-arrow-size: 0;">
      <sl-button>No Arrow</sl-button>
  </sl-tooltip>
tooltip에 html 넣어주기 - tooltip 안에 slot="content"인 div를 넣어준다.
I'm not just a tooltip, I'm a tooltip with HTML!
Hover me 

  <sl-tooltip>
      <div slot="content">I'm not <strong>just</strong> a tooltip, I'm a <em>tooltip</em> with HTML!</div>
    
      <sl-button>Hover me</sl-button>
  </sl-tooltip>
tooltip에 max-width 주기 style="--max-width" 사용
Hover me 

  <sl-tooltip style="--max-width: 80px;" content="This tooltip will wrap after only 80 pixels.">
      <sl-button>Hover me</sl-button>
  </sl-tooltip>
overflow에서도 보이도록 하려면 hoist 속성을 사용
No Hoist Hoist 

  <div class="tooltip-hoist" style="position: relative;
  border: solid 2px var(--sl-panel-border-color);
  overflow: hidden;
  padding: var(--sl-spacing-medium);">
      <sl-tooltip content="This is a tooltip">
        <sl-button>No Hoist</sl-button>
      </sl-tooltip>
    
      <sl-tooltip content="This is a tooltip" hoist>
        <sl-button>Hoist</sl-button>
      </sl-tooltip>
  </div>
